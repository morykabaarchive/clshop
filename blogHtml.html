INTRODUCTION

HTML est le langage de balisage standard pour la création de pages Web. Qu'est-ce que le HTML ? HTML signifie Hyper Text Markup Language HTML est le langage de balisage standard pour la création de pages Web HTML décrit la structure d'une page Web HTML se compose d'une série d'éléments Les éléments HTML indiquent au navigateur comment afficher le contenu Les éléments HTML étiquettent des éléments de contenu tels que "ceci est un titre", "ceci est un paragraphe", "ceci est un lien", etc.

Le HTML (HyperText Markup Language) est un langage de description qui vous permet de décrire l’aspect d’un document, d’y inclure des Informations (Texte, Image, Son, Animation). Le langage a été mis au point par Tim Bernners Lee à partir de 1989.

Le principe du web :

Le web est composé de page web stockées sur des serveurs web, c’est-à-dire des machines connectées à internet en permanence et chargées de fournir les pages web demandées aux machines clientes.
Chacune des pages web est repérée par une adresse appelée URL.

Les éditeurs de textes :

Un éditeur est un logiciel conçu pour rédiger et modifier les documents écrits en HTML.
Exemple : Bloc note, Notepad++, SublimText, etc…

Les navigateurs :

Un navigateur internet est un logiciel informatique qui permet d’interpréter une page HTML et à un utilisateur d’accéder à l’internet.
Exemple : Mozilla Firefox, Google chrome, Opera, etc…

LES FONDATIONS DU HTML5

Les balises :

Les balises permettent d’indiquer la façon dont doit être présenté le document et les liens qu’il établit avec d’autres documents. Elles sont délimitées par le chevron ouvrant (<) et le chevron fermant (>). En règle générale, à toute balise d’ouverture correspond une balise de fermeture.
Exemple : <a> <b> <c> - - - </c> <b> </a> Il a des exceptions, certaines balises dites uniques ne comportent pas de balise de fermeture. Ce type de balise se termine toujours par un slash " / " , mais cette fois le slash se trouve à la fin de la balise.
Exemple : <m/>

Les attributs :

Un attribut est une instruction contenue à l’intérieur de la balise ouvrante et dont le but est de fournir une information supplémentaire sur la manière dont cette balise doit être interprétée.

La déclaration <!DOCTYPE> représente le type de document et aide les navigateurs à afficher correctement les pages Web.La déclaration <!DOCTYPE> n'est pas sensible à la casse.Les titres HTML sont définis avec les balises <h1> à <h6>. <h1> définit l'en-tête le plus important. <h6> définit le titre le moins important :Exemples de base HTML Dans ce chapitre, nous montrerons quelques exemples HTML de base.Ne vous inquiétez pas si nous utilisons des balises que vous n'avez pas encore apprises.Tous les documents HTML doivent commencer par une déclaration de type de document : <!DOCTYPE html>.Le document HTML lui-même commence par <html> et se termine par </html>.La partie visible du document HTML se situe entre <body> et </body>.

La structure générale d’une page HTML

<!DOCTYPE html>
<html>
<head>
<title></title>
<meta charset="utf-8"/>
</head>
<body>

</body>
</html>

Explication :

  1. <!DOCTYPE html> : Cette balise sert à dire au navigateur que ce que vous faites est une page HTML5.
  2. <html> - - - </html> : c’est la balise principale qui englobera toute votre page HTML5. Comme vous pouvez le voir, on ne la ferme qu’en dernier avec </html>,
    qui indique que votre page HTML5 s’arrête là.
  3. <head> - - - </head> : Cette balise contient quelques informations d’en-tête pour votre page web. A l’intérieur de la balise <head>, vous trouvez les balises :
  4. <title> - - - </title> : Cette balise affiche le titre de la page dans la barre de titre du navigateur.
  5. <meta charset="UTF-8"> : Cette balise sert à indiquer que vous allez taper des caractères spéciaux.
  6. <body> - - -</body> : Cette balise contient le contenu de votre page web.

Les Liens

Les liens hypertextes sont des éléments d’une page HTML (souligné lorsqu’il s’agit d’un texte) qui emmènent dans un autre endroit lorsqu’on clique dessus. C’est ce qui permet de lier des pages web entre elles. Les liens hypertextes permettent de naviguer :

  • Vers un autre endroit du document.
  • Vers un fichier HTML situé à un emplacement.
  • Vers un site web.
Lien vers un fichier :

<a href="fichier.html"> - - - </a>

Lien vers un site web :

<a href= "http:// www.nom_site.com"> - - -

Lien vers une ancre :

Une ancre est un point de repère que vous pouvez mettre dans vos pages HTML lorsqu’elles sont très longues. Pour créer une ancre, il suffit de rajouter l’attribut id
à une balise qui va servir de repère. Ça peut être n’importe quelle balise. Ensuite, il suffit de créer un lien comme d’habitude, mais cette fois l’attribut href contiendra
un dièse (#) suivi du nom de l’ancre.
  • Créer le point de repère :
<balise id="nom_ancre"> - - - </balise>
  • Lien vers l’ancre :
<a href="#nom_ancre"> - - -</a>
  • Un lien qui affiche une infobulle au survol :
Utilisez l’attribut "title" qui affiche une bulle d’aide lorsqu’on pointe sur le lien. <a href=" - - - " title=" texte"> - - - </a>
  • Un lien qui ouvre une nouvelle fenêtre :
Il est possible de forcer l’ouverture d’un lien dans une nouvelle fenêtre. Pour cela, on rajoutera l’attribut target= "_blank". <a href=" - - - " target=" _blank"> - - - </a>
NB : utiliser cet attribut pour un lien qui affiche un site web. Il est déconseillé d’utiliser cette technique pour les liens internes de votre site.
